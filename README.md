# little-cms

## Overview

This is a custom CMS platform for managing orders, reservations, and the menu for the [Little Lemon Restaurant](https://github.com/David-Werth/little-lemon).

**Technologies Used:**
- Next.js
- TypeScript
- MongoDB
- Tailwind.css
